## Research Focus

The JOTP research program focuses on:

1. **Formal Equivalence**: Proving Java 26 + Virtual Threads ≡ Erlang/OTP
2. **Empirical Validation**: Real-world testing through enterprise patterns
3. **Phase Change Technology**: Understanding the paradigm shift in concurrent programming
4. **Pattern Mapping**: Systematic mapping of OTP patterns to Java 26

## Usage

These documents are primarily for:
- Academic researchers studying concurrent programming paradigms
- Graduate students researching formal methods and language design
- Industry practitioners interested in the theoretical foundations
- Anyone interested in deep technical analysis of OTP patterns
