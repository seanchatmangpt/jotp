# Distributed System Issues Troubleshooting Guide

import { Tabs } from '@nextra/components'

## Overview

This guide covers troubleshooting distributed JOTP systems including cluster formation issues, network partitions, node failures, leader election problems, and distributed state management.

<Callout type="warning">
  Distributed system failures can cause data inconsistency. Always understand the consistency model before attempting recovery.
</Callout>

## Cluster Formation Issues

### Symptoms

- Nodes fail to discover each other
- `DistributedNode` startup timeout
- Nodes start in standalone mode
- No leader elected
- Connection refused errors

### Common Causes Checklist

- [ ] Network firewall blocking ports
- [ ] Incorrect hostnames/IPs in config
- [ ] DNS resolution failures
- [ ] Clock skew between nodes
- [ ] Different JOTP versions
- [ ] Mismatched node names

## Network Partitions

### Symptoms

- Nodes report different cluster members
- Multiple leaders elected
- Inconsistent state across nodes
- Split-brain scenarios
- Requests timeout or return stale data

### Common Causes Checklist

- [ ] Network switch failure
- [ ] Firewall rule changes
- [ ] Network congestion
- [ ] Partial network failure
- [ ] Clock synchronization issues
- [ ] Network partition tolerance not configured

### Diagnostic Steps

#### 1. Detect Network Partition

```java
// Monitor for network partition
public class PartitionMonitor {

    private final DistributedNode node;
    private final Set<NodeId> lastKnownCluster = new HashSet<>();

    @Scheduled(fixedRate = 5000)
    public void checkPartition() {
        Set<NodeId> currentCluster = node.getClusterMembers();

        // Check if cluster size changed significantly
        if (Math.abs(currentCluster.size() - lastKnownCluster.size()) >
            lastKnownCluster.size() / 2) {

            log.error("Potential network partition detected!");
            log.error("Previous cluster: {}", lastKnownCluster);
            log.error("Current cluster: {}", currentCluster);

            alerting.sendAlert("Network partition detected",
                "previous", lastKnownCluster.toString(),
                "current", currentCluster.toString());
        }

        lastKnownCluster.clear();
        lastKnownCluster.addAll(currentCluster);
    }
}
```

#### 3. Monitor Clock Skew

```java
// Check clock synchronization
@Scheduled(fixedRate = 30000)
public void checkClockSkew() {
    Instant localTime = Instant.now();
    Map<NodeId, Duration> skews = new HashMap<>();

    for (NodeId clusterNode : node.getClusterMembers()) {
        try {
            Instant remoteTime = askNodeForTime(clusterNode);
            Duration skew = Duration.between(localTime, remoteTime);
            skews.put(clusterNode, skew);

            if (Math.abs(skew.toMillis()) > 1000) {
                log.warn("Clock skew > 1s with {}: {}", clusterNode, skew);
            }
        } catch (Exception e) {
            log.error("Failed to get time from {}", clusterNode);
        }
    }

    Duration maxSkew = skews.values().stream()
        .map(Duration::toMillis)
        .map(Math::abs)
        .max(Long::compare)
        .map(Duration::ofMillis)
        .orElse(Duration.ZERO);

    if (maxSkew.toMillis() > 5000) {
        log.error("Excessive clock skew detected: {}", skews);
        alerting.sendAlert("Clock skew too high", "max_skew_ms", maxSkew.toMillis());
    }
}
```

### Solutions

#### Solution 1: Implement Quorum-Based Decisions

```java
// Only proceed if quorum agrees
public class QuorumBasedCoordinator {

    private final int quorumSize;

    public boolean achieveQuorum(Supplier<Boolean> operation) {
        int successes = 0;
        int failures = 0;
        int required = quorumSize;

        for (NodeId node : clusterMembers) {
            try {
                Boolean result = askNode(node, operation);
                if (result) {
                    successes++;
                    if (successes >= required) {
                        return true;  // Quorum achieved
                    }
                }
            } catch (Exception e) {
                failures++;
                if (failures >= quorumSize) {
                    return false;  // Quorum impossible
                }
            }
        }

        return false;
    }
}
```

#### Solution 2: Implement Heartbeat Monitoring

```java
// Monitor node liveness
public class HeartbeatMonitor {

    private final Map<NodeId, Instant> lastHeartbeat = new ConcurrentHashMap<>();
    private final Duration heartbeatTimeout = Duration.ofSeconds(10);

    @Scheduled(fixedRate = 1000)
    public void checkHeartbeats() {
        Instant now = Instant.now();

        lastHeartbeat.entrySet().removeIf(entry -> {
            Duration elapsed = Duration.between(entry.getValue(), now);
            if (elapsed.compareTo(heartbeatTimeout) > 0) {
                log.warn("Node {} missed heartbeat, last seen {}s ago",
                    entry.getKey(), elapsed.getSeconds());

                // Notify cluster of suspected failure
                notifySuspectedFailure(entry.getKey());
                return true;  // Remove from map
            }
            return false;
        });
    }

    public void recordHeartbeat(NodeId node) {
        lastHeartbeat.put(node, Instant.now());
    }
}
```

#### Solution 3: Implement Partition Recovery

```java
// Handle partition recovery
public class PartitionRecoveryHandler {

    public void onPartitionHealed() {
        log.info("Partition healed, starting recovery");

        // 1. Determine who has latest state
        Map<NodeId, Long> vectorClocks = new HashMap<>();
        for (NodeId node : clusterMembers) {
            try {
                Long clock = askNodeForVectorClock(node);
                vectorClocks.put(node, clock);
            } catch (Exception e) {
                log.error("Failed to get vector clock from {}", node);
            }
        }

        // 2. Identify node(s) with latest state
        Long maxClock = vectorClocks.values().stream()
            .max(Long::compare)
            .orElse(0L);

        List<NodeId> latestNodes = vectorClocks.entrySet().stream()
            .filter(entry -> entry.getValue().equals(maxClock))
            .map(Map.Entry::getKey)
            .toList();

        if (latestNodes.size() > 1) {
            log.warn("Multiple nodes with same vector clock: {}", latestNodes);
            // Need conflict resolution
            resolveConflicts(latestNodes);
        } else {
            // Sync from latest node
            syncState(latestNodes.get(0));
        }

        // 3. Resynchronize cluster
        resynchronizeCluster();
    }
}
```

## Node Failures

### Symptoms

- Node stops responding
- Leader election triggered
- Applications failover
- Increased load on remaining nodes
- Data inconsistency

### Common Causes Checklist

- [ ] Hardware failure
- [ ] OutOfMemoryError
- [ ] Network connectivity loss
- [ ] Process crash
- [ ] OS crash
- [ ] Resource exhaustion

### Diagnostic Steps

#### 1. Detect Node Failure

```java
// Monitor node health
public class NodeHealthMonitor {

    private final Map<NodeId, Instant> lastSeen = new ConcurrentHashMap<>();
    private final Duration failureTimeout = Duration.ofSeconds(30);

    @Scheduled(fixedRate = 5000)
    public void checkNodeHealth() {
        Instant now = Instant.now();

        for (NodeId node : clusterMembers) {
            if (!node.equals(selfId)) {
                Duration elapsed = Duration.between(
                    lastSeen.getOrDefault(node, now.minus(Duration.ofHours(1))),
                    now
                );

                if (elapsed.compareTo(failureTimeout) > 0) {
                    log.error("Node {} suspected failure, last seen {}s ago",
                        node, elapsed.getSeconds());

                    // Confirm failure
                    if (confirmFailure(node)) {
                        handleNodeFailure(node);
                    }
                }
            }
        }
    }

    private boolean confirmFailure(NodeId node) {
        // Try multiple times to confirm
        for (int i = 0; i < 3; i++) {
            try {
                ping(node, Duration.ofSeconds(2));
                return false;  // Node is alive
            } catch (Exception e) {
                log.warn("Ping attempt {} failed for {}", i + 1, node);
            }
        }
        return true;  // Node is dead
    }
}
```

```java
// Implement graceful shutdown
public class GracefulShutdown {

    @PreDestroy
    public void shutdown() {
        log.info("Starting graceful shutdown");

        // 1. Stop accepting new work
        loadBalancer.drain();

        // 2. Notify cluster we're leaving
        broadcastShutdownNotification();

        // 3. Wait for in-flight work to complete
        waitForWorkCompletion(Duration.ofMinutes(2));

        // 4. Trigger failover for leadership
        if (amLeader()) {
            triggerFailover();
        }

        // 5. Close connections
        closeConnections();

        // 6. Shutdown processes
        ProcRegistry.getInstance().shutdown();
```

#### Solution 4: Data Replication

```java
// Replicate data across nodes
public class DataReplicator {

    private final int replicationFactor = 3;

    public void write(String key, byte[] value) {
        // Write to local node
        localStore.put(key, value);

        // Replicate to other nodes
        List<NodeId> replicas = selectReplicas(key, replicationFactor);

        CountDownLatch latch = new CountDownLatch(replicas.size());
        AtomicInteger successes = new AtomicInteger(0);

        for (NodeId replica : replicas) {
            CompletableFuture.runAsync(() -> {
                try {
                    replicateTo(replica, key, value);
                    successes.incrementAndGet();
                } catch (Exception e) {
                    log.error("Replication failed to {}", replica, e);
                } finally {
                    latch.countDown();
                }
            });
        }

        try {
            // Wait for replication to complete
            boolean completed = latch.await(5, TimeUnit.SECONDS);

            if (!completed || successes.get() < replicationFactor) {
                log.warn("Replication incomplete: {}/{} succeeded",
                    successes.get(), replicationFactor);
            }
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
    }
}
```

```java
// Monitor leader elections
public class ElectionMonitor {

    private NodeId lastLeader = null;
    private Instant lastElectionTime = null;
    private final AtomicInteger electionCount = new AtomicInteger(0);

    @EventListener
    public void onLeaderElection(LeaderElectedEvent event) {
        NodeId newLeader = event.newLeader();
        Instant now = Instant.now();

        log.info("Leader elected: {}", newLeader);

        if (lastLeader != null && !lastLeader.equals(newLeader)) {
            // Leader changed
            electionCount.incrementAndGet();

            if (lastElectionTime != null) {
                Duration elapsed = Duration.between(lastElectionTime, now);
                if (elapsed.getSeconds() < 60) {
                    log.warn("Frequent leader changes: {} in {}s",
                        electionCount.get(), elapsed.getSeconds());
                    alerting.sendAlert("Frequent leader changes",
                        "count", String.valueOf(electionCount.get()));
                }
            }
        }

        lastLeader = newLeader;
        lastElectionTime = now;
    }
}
```

```java
// Verify priority configuration
public class PriorityVerifier {

    public void verifyConfiguration() {
        List<DistributedAppSpec> specs = distributedNode.getAppSpecs();

        for (DistributedAppSpec spec : specs) {
            log.info("App: {}", spec.name());
            log.info("  Priority groups:");

            NodeId leader = distributedNode.getLeader(spec.name());
            if (leader != null) {
                boolean inList = spec.nodes().stream()
                    .flatMap(List::stream)
                    .anyMatch(node -> node.equals(leader));
```

#### Solution 2: Handle Split-Brain

```java
// Detect and resolve split-brain
public class SplitBrainResolver {

    public void resolveSplitBrain() {
        // 1. Identify all nodes claiming leadership
        List<NodeId> claimLeaders = new ArrayList<>();
        for (NodeId node : clusterMembers) {
            try {
                if (isLeader(node)) {
                    claimLeaders.add(node);
                }
            } catch (Exception e) {
                log.error("Failed to check leadership of {}", node);
            }
        }

        if (claimLeaders.size() > 1) {
            log.error("Split-brain detected: {} leaders", claimLeaders.size());

            // 2. Use deterministic resolution
            NodeId winner = claimLeaders.stream()
                .min(Comparator.comparing(NodeId::name))  // Lowest name wins
                .orElseThrow();

            log.info("Resolving split-brain, winner: {}", winner);

            // 3. Tell losers to step down
            for (NodeId node : claimLeaders) {
                if (!node.equals(winner)) {
                    tellNodeToStandDown(node);
                }
            }
        }
    }
}
```

```java
        latch = new LeaderLatch(zkClient, "/election/payment-service");

        latch.addListener(new LeaderLatchListener() {
            @Override
            public void isLeader() {
                log.info("Became leader!");
                onBecameLeader();
            }

            @Override
            public void notLeader() {
                log.info("No longer leader");
                onLostLeadership();
            }
        });

        latch.start();
    }

    public boolean amLeader() {
        return latch.hasLeadership();
    }
}
```

#### Solution 4: Add Leadership Timeout

```java
// Add timeout for leader inactivity
public class LeadershipTimeout {

    private final Duration timeout = Duration.ofSeconds(30);
    private volatile Instant lastLeaderActivity = Instant.now();

    @Scheduled(fixedRate = 5000)
    public void checkLeaderActivity() {
        if (distributedNode.amLeader()) {
            lastLeaderActivity = Instant.now();
        } else {
            Duration elapsed = Duration.between(
                lastLeaderActivity,
                Instant.now()
            );

            if (elapsed.compareTo(timeout) > 0) {
                log.error("Leader inactive for {}s, triggering reelection",
                    elapsed.getSeconds());
                triggerElection();
            }
        }
    }
}
```

### Prevention Strategies

- Configure clear priority order
- Use distributed coordination for critical systems
- Monitor leader election frequency
- Implement split-brain detection
- Add leadership timeouts
- Test leader election in staging
- Use odd number of nodes for elections

   ```java
   distributedNode.getClusterMembers().forEach(node -> {
   ```

   - Ping results between all nodes
   - Firewall rules
   - Network latency measurements

   - NodeId configurations
   - Priority lists
   - Timeout values
