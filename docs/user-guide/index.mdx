# JOTP User Guide

Welcome to the comprehensive JOTP documentation, organized using the **Diátaxis framework** for clear, focused documentation.

## What is JOTP?

**JOTP** is a production-ready Java 26 framework implementing all 15 OTP (Erlang/OTP) primitives. It brings battle-tested concurrency patterns, supervision trees, and fault tolerance to the JVM using virtual threads, sealed types, and pattern matching.

### Key Features

- **Lightweight Processes**: Millions of concurrent processes with virtual threads
- **Fault Tolerance**: "Let it crash" philosophy with automatic recovery
- **Supervision Trees**: Hierarchical process management with restart strategies
- **Type Safety**: Sealed types ensure exhaustive message matching
- **Event-Driven**: Typed pub/sub event bus for loose coupling
- **Distribution**: Location-transparent actor messaging

## Quick Start

### 1. Create a Process

```java
// Define state and messages
record CounterState(int value) {}
sealed interface CounterMessage permits CounterMessage.Increment {}
record Increment() implements CounterMessage {}

// Spawn process
Proc<CounterState, CounterMessage> proc = Proc.spawn(
    "counter",
    new CounterState(0),
    (state, msg) -> switch (msg) {
        case Increment() -> new CounterState(state.value() + 1);
    }
);

// Send message
proc.send(new Increment());
```

### 2. Create a Supervisor

```java
Supervisor supervisor = Supervisor.spawn(
    "my-supervisor",
    Strategy.ONE_FOR_ONE,
    List.of(
        ChildSpec.of(
            "worker-1",
            () -> WorkerProc.spawn(),
            RestartStrategy.PERMANENT,
            Duration.ofSeconds(5)
        )
    )
);
```

### 3. Use Event Bus

```java
EventManager<DomainEvent> bus = EventManager.spawn("events");

// Subscribe
bus.subscribe("handler", event -> {
    System.out.println("Received: " + event);
    return SubscriberState.KEEP;
});

// Publish
bus.publish(new OrderCreated("ORD-123", BigDecimal.valueOf(99.99)));
```

## The 15 OTP Primitives

| Primitive | Purpose | API |
|-----------|---------|-----|
| **Proc** | Lightweight process with mailbox | `Proc<S,M>.spawn()` |
| **Supervisor** | Fault-tolerant process tree | `Supervisor.spawn()` |
| **StateMachine** | State machine with events | `StateMachine<S,E,D>.spawn()` |
| **ProcRef** | Stable process reference | `ProcRef.of()` |
| **ProcMonitor** | One-way death notification | `ProcMonitor.monitor()` |
| **ProcLink** | Bidirectional crash propagation | `ProcLink.link()` |
| **ProcRegistry** | Name-based process lookup | `ProcRegistry.lookup()` |
| **ProcTimer** | Scheduled message delivery | `ProcTimer.send()` |
| **ProcSys** | Live process introspection | `ProcSys.suspend()` |
| **ProcLib** | Process utility functions | `ProcLib.init_ack()` |
| **CrashRecovery** | Isolated execution wrapper | `CrashRecovery.supply()` |
| **Parallel** | Structured concurrency | `Parallel.run()` |
| **EventManager** | Typed pub-sub events | `EventManager<E>.spawn()` |
| **Result** | Railway-oriented errors | `Result.ok() / Result.err()` |
| **ExitSignal** | Exit reason carrier | `ExitSignal.exception()` |
