# Getting Started with JOTP

Welcome to JOTP! This tutorial will guide you through setting up your development environment and running your first fault-tolerant concurrent application.

## What You'll Learn

- Setting up a Java 26 development environment
- Understanding the JOTP project structure
- Running your first Proc-based application
- Basic testing patterns

## Prerequisites

- Java 26 JDK with preview features enabled
- Maven 4.0+ (or mvnd - Maven Daemon)
- Basic familiarity with Java concurrency concepts

## Outline

### 1. Environment Setup
- Installing Java 26
- Configuring Maven with preview flags
- Verifying your installation

### 2. Project Structure Overview
- Understanding the module system
- Core packages and their purposes
- Test organization

### 3. Hello World Process
- Creating your first `Proc<String, String>`
- Understanding message passing
- Running and testing

### 4. Next Steps
- Building supervisor trees
- Event-driven patterns
- Distributed actors

## Related Tests
- `ProcTest.java` - Basic process lifecycle
- `ApplicationTest.java` - Application initialization
- `EventManagerTest.java` - Event broadcasting basics
